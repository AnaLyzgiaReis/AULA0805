Fundamentos básicos da linguagem C

  1. Estrutura básica de um programa em C

    2. Para começar um codigo é nescessario haver uma estrutura de codigo obrigadoria:

      #include <stdio.h> // Biblioteca para entrada/saída

      int main() { // Função principal

      return 0; // Retorno da função main

      }

  2. Tipos de Dados e Variáveis

    3. Os tipos de variaveis disponivel em C são:

      int idade = 25 Inteiro
      float altura = 1.75 Ponto flutuante
      char letra = 'A' Caractere
      double pi = 3.141592 Ponto flutuante de dupla precisão

  3. Tipos de Operadores

    4. Os tipos/sinais disponivel em C são:

      Aritméticos +, -, *, /, %
      Relacionais ==, !=, >,<
      Lógicos && (E); || (OU) e ! (NÃO/DIFERENTE)
